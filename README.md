# Create Webpack App

## Getting Started

### Installation

Install globally

```bash
npm install -g create-webpack-app
```

### Create app

```bash
create-webpack-app my-app
cd my-app
```

Create a directory called my-app with the initial structure of the project.
Will also install its dependencies.

```
my-app
├── node_modules
├── package.json
├── public
│   └── index.html
└── src
    └── entry.js
```

### Scripts

```bash
npm start
npm build
```
